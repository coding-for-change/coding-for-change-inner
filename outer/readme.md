# codingforchange.com

This is one of two repositories created for my codingforchange.com website <a href="codingforchange.com"><samp>codingforchange.com</samp></a>. Forked from Henry Hefferman

<br>

To setup a dev environment:

```bash
# Clone the repository

# Install dependencies 
npm i

# Run the local dev server
npm run dev
```

To serve a production build:

```bash
# Install dependencies if not already done - 'npi i'

# Build for production
npm run build

# Serve the build using express
npm start
```
